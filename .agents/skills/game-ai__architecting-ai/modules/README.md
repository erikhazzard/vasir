# Genre Modules

Use the core skill first. Then load the relevant modules below.

- `territory_arena.md` — Paper.io-like, Tron/Snake-like, capture loops, arena territory control.
- `combat_shooter.md` — enemies, squads, cover, line of sight, flanking, bosses.
- `sports_vr.md` — sports bots, VR opponents/teammates, body-scale movement, possession, spacing.
- `companion_social.md` — companions, helpers, social NPCs, dialogue-adjacent behavior.
- `strategy_tactics.md` — RTS/tactics, squads, formations, fog of war, map control.
- `racing_chase.md` — racing, pursuit, evasion, vehicles, rubber-banding.
- `simulation_economy.md` — sim, economy, management, progression, resource agents.
- `autoplay_testing.md` — development agents for QA, tuning, exploit search, telemetry.

If multiple modules apply, combine them. Example: a VR basketball defender uses `sports_vr.md`, `director_layer.md`, and possibly `autoplay_testing.md` for tuning.
