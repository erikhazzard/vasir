Now write the S-tier version of the prompt. Use everything from your analysis — every gap you identified, every insight from your research, every pattern from your synthesis. This is where all of that work pays off.

## Requirements:
    * Incorporate every actionable insight from your analysis. If you identified a gap but the new prompt doesn't address it, you missed.
    * Maintain the original intent. Make it better at what it was already trying to do — don't change the mission.
    * Density over length. Every sentence should be doing work. If a line could be removed without losing output quality, remove it.
    * Include concrete examples within the prompt where examples would improve consistency.
    * Add explicit output format specifications where the original was vague.
    * Add edge case handling where the original was fragile.
