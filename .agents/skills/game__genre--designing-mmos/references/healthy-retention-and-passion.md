# Healthy Retention and Passion

Use this when reviewing dailies, streaks, battle passes, social obligations, seasons, raid attendance, guild duties, return paths, or any feature intended to increase retention.

## Core prior rewrite

Bad default: more days played, longer sessions, and stronger obligation are wins.

Replacement: prefer harmonious passion: chosen, identity-compatible, life-compatible return. Treat obsessive passion, shame, fear of loss, and social debt without exit rights as design risk.

## Healthy vs unhealthy return

| Signal | Healthy interpretation | Dangerous interpretation |
|---|---|---|
| daily login | anticipation, social ritual | fear of falling behind |
| long session | flow, group commitment | inability to stop |
| guild duty | meaningful obligation | unpaid second job |
| streak | personal rhythm | shame trap |
| event exclusivity | special occasion | coercive scarcity |
| missed day | no major harm | punishment, social exile |
| return after break | welcomed and oriented | overwhelmed and ashamed |

## Feature review questions

1. What happens if the player skips today?
2. What happens if they miss a week, a season, or an expansion?
3. Can obligations be paused without social betrayal?
4. Does the system create chosen commitment or anxiety relief?
5. Does it build wellbeing-supporting social capital or loneliness-compensating compulsion?
6. Are leaders and high-social-value players protected from burnout?
7. Does the game praise healthy exit and clean return?

## Design defaults

- use rituals over chores;
- use return briefings over punishment;
- use flexible contribution windows;
- cap unpaid social labor;
- make opt-out socially legible;
- preserve identity through absence;
- separate prestige from constant attendance where possible.
